第一课 
================

本篇内容是为初学者准备的知识，内容包括：

**课前预习：**

#. 预习本课件第2，3章节

**课堂讲解：**

#. 掌握在线构建 spring boot 项目
#. 掌握 maven 环境，仓库知识
#. 掌握 maven 的依赖关系
#. 掌握 maven 的执行周期
#. 掌握 git 的使用方法
#. 了解 grade 的使用方法

**课后练习：**

#. 准备自己的开发环境；
#. 完成练习：https://github.ibm.com/fsd-training/spring-boot-course/tree/master/assignment/session1
#. 部署自己的练习作品到服务器上；
#. 更新学员练习进度表：https://pages.github.ibm.com/fsd-training/spring-boot-course/



第二课 
================

如果你完成了第一课的练习，请继续下面的内容。 本篇内容包括：

**课前预习：**

#. 预习本课件第5，6，7章节

**课堂讲解：**

#. 掌握Spring Boot 项目基本的CRUD(Create Read Update Delete)操作

**课后练习：**

#. 搭建Spring Boot项目开发环境，添加依赖（Web Thymeleaf JPA H2）；
#. 完成练习：https://github.ibm.com/lanzejun/q3-spring-cloud/tree/master/session2
#. 部署自己的练习作品到服务器上；
#. 更新学员练习进度表：https://pages.github.ibm.com/fsd-training/spring-boot-course/
